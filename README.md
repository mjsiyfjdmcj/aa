# Brighters - Climate Action Non-Profit Website

A responsive website for a climate-focused non-profit organization dedicated to environmental conservation and sustainable development.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Mobile Navigation**: Hamburger menu for mobile devices with smooth transitions
- **Smooth Scrolling**: Enhanced user experience with smooth scroll navigation
- **Modern UI**: Clean and professional design with climate-themed color scheme
- **Multiple Sections**: 
  - Hero section with call-to-action
  - Statistics showcase
  - About section
  - Programs overview
  - Impact highlights
  - Contact footer

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid layouts
- **JavaScript**: Interactive navigation and smooth scrolling

## File Structure

```
brighters.org/
│
├── index.html          # Main HTML file
├── style.css           # Stylesheet
├── script.js           # JavaScript functionality
└── README.md           # Documentation
```

## Setup Instructions

1. Clone or download the project files
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Customization

### Colors
The main color scheme uses:
- Primary Green: `#00a86b`
- Accent Orange: `#ffa500`
- Dark Background: `#2c3e50`

### Images
Images are sourced from Unsplash. Replace the URLs in `index.html` with your own images:
- Hero background
- About section image
- Impact section image

### Content
Edit the text content in `index.html` to match your organization's mission and programs.
